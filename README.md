# 🎓 Student Management System

Python를 활용하여 만든 학생 성적 관리 프로그램입니다.

## 📌 프로젝트 소개

학생 정보를 관리하고 성적을 조회하는 콘솔 프로그램입니다.

## ✨ 주요 기능

- 학생 등록
- 학생 조회
- 학생 수정
- 학생 삭제
- 성적 조회

## 🛠 사용 기술

- Python
- Git
- GitHub

## 📚 자료구조

- List
- Dictionary

## 🚀 실행 방법

```bash
python main.py
```
